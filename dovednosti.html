Dovednosti

HTML

Zvládám základy HTML pro tvorbu jednoduchých webových stránek. Umím strukturovat obsah pomocí značek a znám základní principy sémantiky. Využívám HTML při vytváření projektů v rámci kurzů a vlastního portfolia.

CSS

Dokážu stylovat stránky pomocí základních CSS pravidel. Pracuji s barvami, rozvržením, texty i box modely. Učím se využívat flexbox a grid pro responzivní design a ladím stylování tak, aby působilo čistě a přehledně.

Python

Python je můj hlavní jazyk, ve kterém se aktivně vzdělávám. Ovládám základy syntaxe, práci s proměnnými, cykly, podmínkami i funkcemi. Postupně pronikám do práce s databázemi, frameworky a objektově orientovaného programování.

SQL

S SQL teprve začínám, ale mám základní přehled o práci s relačními databázemi. Umím vytvářet jednoduché dotazy pro výběr dat pomocí příkazu SELECT a chápu základní principy struktury tabulek. Postupně se učím další možnosti filtrování a práci s více tabulkami.
